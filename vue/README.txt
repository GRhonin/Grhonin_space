项目描述：仿ThinkPad移动端官网
架构：1：vue + vue-cli + axios + vuex + vue-router
	  2：使用vue+webpack构建项目环境
	  3：使用node（express框架）+ mysql搭建后台服务器
	  4：使用node安装第三方插件
功能模块：
首页：轮播图使用了swiper，图片内容是在首页组件的created里面发送ajax请求服务器所得；下面相同样式的模块同理；头部下拉动画直接用vue自带的transition标签
登录/注册：在进入个人中心、购物车等需要登录的页面时，使用cookie和路由守卫进行判断及重定向， 登录成功后用cookie保存用户名并返回相应的页面。
官方商城：中间部分的商品及轮播数据依旧是从服务器拿来的。中间部分商品点击可跳转至其对应的详情页。
	商品详情：主要有商品展示、购物车入口，活动倒计时，加入购物车功能
购物车：主要功能是购物车商品的增删改，核心是购物车列表的id和数据返回后加入了check属性。通过这两点来达到选择商品进行增删改及计算价格等功能
个人中心：入口是首页（登陆后登录按钮会被替换为用户名，点击可以跳转到个人中心）和商品详情的头部=>个人中心；主要功能：查看订单及地址增删改查。
